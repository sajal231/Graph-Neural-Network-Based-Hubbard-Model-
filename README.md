# Quantum-ML-Energy-Prediction

This repository contains a classical machine learning approach using Graph Neural Networks (PyTorch + PyTorch Geometric)
to predict quantum potential energy curves for the Hubbard chain model.

## Project Overview
- Input: On-site interaction parameter (U/t)
- Output: Ground-state energy per site
- Model: Graph Neural Network (PyTorch Geometric)
- Weighted MSE loss for edge points

## Features
- Periodic 1D Hubbard chain
- Positional encoding for nodes
- Custom weighted dataset
- Training/Validation split
- Prediction and plotting

## Applications
- Quantum Simulation
- Computational Physics
- Quantum Machine Learning
